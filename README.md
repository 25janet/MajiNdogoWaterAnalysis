#  Maji Ndogo Water Access Analysis

## Project Overview

The **Maji Ndogo Water Access Analysis** project focuses on analyzing water access data to uncover insights about availability, distribution, and disparities across regions.  
The project demonstrates a **complete data analysis workflow**, using **Microsoft Excel** for data preparation and **Power BI** for visualization and insights.

This analysis supports data-driven decision-making related to water access and infrastructure planning.

---

## Objectives

- Clean and prepare raw water access data
- Analyze water availability and access patterns
- Identify trends, gaps, and key metrics
- Build an interactive dashboard for easy interpretation of insights

---

## Tools & Technologies Used

###  Microsoft Excel
Excel was used during the **data preparation and exploration phase**:

- Data cleaning and formatting
- Handling missing and inconsistent values
- Basic calculations and derived columns
- Initial data exploration and validation
- Preparing the dataset for visualization

---

###  Power BI
Power BI was used for **data modeling, analysis, and visualization**:

- Importing and modeling cleaned Excel data
- Creating relationships and measures
- Designing interactive dashboards
- Visualizing trends using charts, tables, and KPIs
- Enabling slicers and filters for dynamic analysis

---

## Analysis Workflow

1. **Data Collection**  
   Raw water access data sourced and reviewed.

2. **Data Cleaning (Excel)**  
   - Removed inconsistencies  
   - Standardized formats  
   - Verified data accuracy  

3. **Data Modeling (Power BI)**  
   - Loaded cleaned data from Excel  
   - Created calculated measures and relationships  

4. **Visualization & Insights (Power BI)**  
   - Built interactive visuals  
   - Highlighted key water access patterns  
   - Enabled user-driven exploration  

---

## Key Insights

- Variations in water access across different regions
- Differences between urban and rural water availability
- Identification of areas with limited or unreliable access
- Data patterns that can inform infrastructure improvement decisions

## Project Outputs

- Interactive Power BI dashboard
